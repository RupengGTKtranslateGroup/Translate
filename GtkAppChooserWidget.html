Description

GtkAppChooserWidget is a widget for selecting applications. It is the main building block for GtkAppChooserDialog. Most applications only need to use the latter; but you can use this widget as part of a larger widget if you have special needs.

GtkAppChooserWidget offers detailed control over what applications are shown, using the â€œshow-defaultâ€, â€œshow-recommendedâ€, â€œshow-fallbackâ€, â€œshow-otherâ€ and â€œshow-allâ€ properties. See the GtkAppChooser documentation for more information about these groups of applications.

To keep track of the selected application, use the â€œapplication-selectedâ€ and â€œapplication-activatedâ€ signals.
