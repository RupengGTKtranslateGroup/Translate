GtkBox versus GtkGrid: spacing

With GtkBox, you have to specify the ‚Äúspacing‚Äù when you construct it. This property specifies the space that separates the children from each other. Additionally, you can specify extra space to put around each child individually, using the GtkBox padding child property.

GtkGrid is very similar when it comes to spacing between the children, except that it has two separate properties, ‚Äúrow-spacing‚Äù and ‚Äúcolumn-spacing‚Äù, for the space to leave between rows and columns. Note that row-spacing is the space between rows, not inside a row. So, if you doing a horizontal layout, you need to set ‚Äúcolumn-spacing‚Äù.

GtkGrid doesn't have any custom child properties to specify per-child padding; instead you can use the ‚Äúmargin‚Äù property. You can also set different padding on each side with the ‚Äúmargin-left‚Äù, ‚Äúmargin-right‚Äù, ‚Äúmargin-top‚Äù and ‚Äúmargin-bottom‚Äù properties.
